QUICK START

  • select list (or "+" to start new)
  • capture image of name(s)
    • ==> names added to list
  • repeat until done
  • select "Get tree"

LISTS. To begin, select a list or "+" to start a new one. Captured names go into the active list. Manage your lists from Home.

CAPTURES. We capture text by OCR, then GNRD finds the biological names. Other text is ignored, e.g., you can photograph a whole book page to get just 1 name if you want.

TREES. To get a tree, you have to be viewing a list. Phylogenetic relationships come from the OpenTree project.

IMAGE QUALITY. Clear obstructions. Hold camera steady, level and perpendicular to the source, 8 to 20 inches away. Activate auto-focus by touching the name.

TROUBLE-SHOOTING. Usually OCR is the problem. Reduce blur, double-image, and glare, and avoid low-contrast sources. If all else fails, write the name neatly on your hand and take a photo of that!

ABOUT. The Phylotastic project aims to satisfy the world's hunger for cheap and fast trees. Please send us your feedback.
