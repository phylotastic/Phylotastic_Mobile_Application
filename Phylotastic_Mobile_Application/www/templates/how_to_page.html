Quick Start
    1. Select list or choose a capture option
    2. Capture name(s)
         ==> names added to list
    3. Repeat until done
    4. Select "Get Tree"
Capture Options:
= Use a stored photo
     = Use text
= Use camera
LISTS.   Create lists from the upper right menu or by saving temporary lists. Captured names go into the active list. Swipe right to delete a name. Manage your lists from Home.
CAPTURES.   We capture text by OCR, then GNRD finds formal names (e.g., "Sus scrofa" not "boar"). Other text is ignored, e.g., you can photograph a whole book page to get just 1 name if you want.
TREES.   To get a tree, you have to be viewing a list. Phylogenetic relationships come from the OpenTree project.
IMAGE QUALITY.   Clear obstructions. Hold camera steady, level and perpendicular to the source, 8 to 20 inches away. Activate auto-focus by touching the name.
TROUBLE-SHOOTING.   Usually OCR is the problem. Reduce blur, double-image, and glare, and avoid low-contrast sources. If all else fails, write the name neatly on your hand and take a photo of that!
FEEDBACK.   Please post feedback including bug reports, comments, and feature requests.
ABOUT.   The Phylotastic project aims to satisfy the world's hunger for cheap and fast trees.
